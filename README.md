# 🎯 C# & .NET Core - Geliştirici Kütüphanesi

Bu depo, C# ve .NET Core ekosisteminde aldığım uzun soluklu eğitimleri, katıldığım kurumsal workshop programlarını, yüksek lisans çalışmalarımı ve geliştirdiğim kişisel projeleri/prototipleri tek bir çatı altında toplayan **ana backend kütüphanemdir**.

Profil kalabalığını önlemek ve geliştirme sürecimi merkezi bir yapıda takip etmek amacıyla **Monorepo** mimarisiyle tasarlanmıştır.

## 📂 Kütüphane Yapısı

Kütüphane içerisindeki klasörlere dair bilgi:

* [BTK-Akademi](/BTK-Akademi) - BTK Akademi platformu üzerindeki C#, .NET Core, Web API eğitimlerine ait proje çıktıları.

* [BookProjects](/BookProjects) - Belirli kitaplardaki kod ve proje uygulamaları.

* [CS_Egitim_Kampi_MY](/CS_Egitim_Kampi_MY) - Murat Yücedağ tarafından yapılan C# Eğitim Kampı kapsamındaki proje uygulamaları.

* [Gazi-FOW](/Gazi-FOW) - Gazi Üniversitesi FOW programı kapsamında gerçekleştirilen teknik çalışmalar ve uygulamalar.

* [MasterDegree](/MasterDegree) - Yüksek lisans (Master) sürecimdeki proje uygulamaları.

* [PatikaDev](/PatikaDev) - Patika.dev platformu üzerindeki C# ve temel .NET patikalarına ait algoritma, pratik ve backend görevleri.

* [TechCareer](/TechCareer) - Techcareer bünyesinde katıldığım, sektörel standartlarda .NET Core backend mimarileri geliştirdiğimiz yoğunlaştırılmış bootcamp/eğitim projeleri.

* [Udemy](/Udemy) - Udemy üzerindeki eğitmenlerden aldığım C# / .NET mimari, tasarım desenleri ve pratik odaklı kurs projeleri.
